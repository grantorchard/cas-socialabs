Lab 02. Cloud Agnostic Blueprinting
***********************************

Solution 01. Assigning Image and Flavor Values
==============================================

.. code-block:: yaml
    :linenos:

    formatVersion: 1
    inputs: {}
    resources:
      Cloud_Machine_1:
      type: Cloud.Machine
      properties:
        image: ubuntu
        flavor: small
        constraints:
        - tag: 'platform:aws'
